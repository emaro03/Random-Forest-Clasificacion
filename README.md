La finalidad del proyecto en cuestión es implementar un modelo de "Random Forest Classifier" para predecir que tipo de flor es según las medidas de los pétalo y sépalo.

### Etapas del proyecto
1. Análisis exploratorio;
2. Implementación de la librería **SKLearn** para el modelado;
3. Implementación del algoritmo **Random Forest Classifier**;
4. Evaluación de la eficiencia del modelo;
5. Obtención del error promedio del modelo.
